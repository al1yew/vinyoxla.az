# vinyoxla.az

Modern Next.js application for VIN report checking and report purchases in Azerbaijan.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- next-intl
- Axios
- Zod
- lucide-react
- shadcn-style local UI primitives

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000/az`.

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```
